Инструкция по установке и настройке
===================================

### 1. Выполняем команды из `Projects/Scripts`:

 Последовательность выполнения команд указана в названии самих `.bash` файлов:

```
1-network-create.bash
2-postgres-run.bash
3-builds.bash
4-webapi-run.bash
5-mvc-run.bash
```
 ---
### 2. После этого, нужно зайти в наш контейнер с `postgres`, для этого используем команду:

```
docker exec -it postgres_db bash
```

После нужно ввести команду для входа в `psql`:
```
psql -U myuser -d mydatabase
```

**-U myuser** - имя пользователя, которое указали при создании контейнера postgres в переменной `POSTGRES_USER`.

**-d mydatabase** - имя базы данных, которое также указали при создании контейнера в переменной `POSTGRES_DB`.

После нужно выполнить команды по `созданию таблиц` и `заполнению их данными`. Сами команды находятся в файле `Projects/Sql`:


```
1_create_tables.sql
2_insert_data.sql
```
---

### 3. Финальный этап - открываем браузер и вводим в адресную строку:
```
http://localhost:8081/Products
```
